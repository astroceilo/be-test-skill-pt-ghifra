# Product Performance API

## 📌 Deskripsi Project
Project ini merupakan backend API untuk sistem analisis performa produk.  
Sistem mengambil data dari dua sumber:
- Web scraping (books.toscrape.com)
- Public API (DummyJSON)

Data kemudian diproses, disimpan ke database SQLite, dan disediakan melalui API untuk digunakan pada dashboard frontend.

---

## ⚙️ Teknologi yang Digunakan
- Node.js
- Express.js
- SQLite
- Axios
- Cheerio (web scraping)
- Nodemon (development)
